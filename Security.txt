Steps to set up tagish
1. Copy tagish.login to ${java.home}/lib/security/
2. Edit java.security and add line
   login.config.url.1=file:${java.home}/lib/security/tagish.login
3. Copy the file lib/NTSystem.dll to ${java.home}/bin folder
4. Set the properties AdminGroup, UserGroup and DomainList in 
   the properties file. Domain list is comma seperated. The first
   value in the list will be selected as default.
